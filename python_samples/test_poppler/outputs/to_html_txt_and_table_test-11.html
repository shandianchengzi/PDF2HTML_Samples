Bits 23:20

L[3:0]

: Regular channel sequence length

These bits are written by software to define the total number of conversions in the regular 
channel conversion sequence.
0000: 1 conversion
0001: 2 conversions 
.....
1111: 16 conversions

SQ16[4:0]

: 16th conversion in regular sequence
